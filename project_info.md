# Pet-проект на Go: система оповещений о ценах

**Стек:** Go (net/http + chi), PostgreSQL, Redis, gocron, Docker

**Что делает:**
- Пользователь добавляет URL товара и желаемую цену
- Проект парсит цену по расписанию (каждые N минут)
- При достижении целевой цены отправляет уведомление (Telegram / email)
- Хранит историю изменений цен в БД

**Почему полезен:**  
Позволяет изучить работу с горутинами, планировщиком задач, HTTP-клиентом, парсингом HTML (goquery), кешированием в Redis и отправкой уведомлений.
